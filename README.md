鈴子ちゃんについて



鈴子ちゃんは、看護師をモチーフとしたオリジナルキャラクターです。白を基調とした看護師風の服装と、紺色の短い髪が特徴。年相応の無邪気な性格と可愛い笑顔が魅力的なキャラクターです。

- 職業：架空の病院 White Wing（白い翼）病院に勤める若手看護師

ローカル開発



このサイトは静的 HTML/CSS のみで構成されています。ローカルで開発する場合：

```bash

git clone https://github.com/abcderp2/suzukochan.officialsite.git

cd suzukochan.officialsite

python -m http.server 8000

```



ブラウザで http://localhost:8000 にアクセスしてください。
