### ローカル開発

このサイトは静的 HTML/CSS のみで構成されています。ローカルで確認する場合：

```bash
git clone [https://github.com/abcderp2/suzukochan.officialsite.git](https://github.com/abcderp2/suzukochan.officialsite.git)
cd suzukochan.officialsite
python -m http.server 8000

```

ブラウザで `http://localhost:8000` にアクセスしてください。
